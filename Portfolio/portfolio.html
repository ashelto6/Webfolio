Ultimate Network

Open-Source Web Development Project in Python

About this project

The UN | May 2020 – Jun 2020

For this project, I used the Python programming language and the Flask Web Development framework. I implemented a slew of Python packages and a few libraries, one being SQLAlchemy, which facilitates the communication between Python programs and databases.

UN, my first web application project was more-so a learning experience as I began my studies of the Python programming language. It's a functional application that allows users to network & connect with one another via direct messages, a lovely follower system and short posts that are visible to users on the site, along with a sleek and simple, but stylish interface to better the user experience.

Through this open-source project I gained tremendous amounts of insight on how to navigate and manage a back-end server/database in the Flask Framework and deploy it to a production server with a private domain.

Cashino

Command Shell Application in C++

About this project

Cashino | Mar 2020 – Apr 2020

Cashino was my idea of a command prompt casino built in C++. My plan was to create a program that gave a user the option to choose between different casino games to play.

Using C++ and OOP, I have implemented the game of Craps into the application and my plans are to continue to implement small games, such as Blackjack as time progresses. I built the application to be scalable and modular so that it will be easy to implement these new games whenever I'd like.
